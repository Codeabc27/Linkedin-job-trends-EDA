
# LinkedIn Job Trends – Exploratory Data Analysis (EDA)

## Overview

This project performs Exploratory Data Analysis (EDA) on LinkedIn job postings data to identify key hiring trends in the technology job market. The analysis focuses on understanding job roles, required skills, experience levels, salary ranges, and location-based demand using Python.

---

## Objectives

* Analyze demand for different job roles
* Identify frequently required technical skills
* Study hiring patterns across locations
* Understand experience and salary distributions

---

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Dataset Description

The dataset includes job-posting information with the following columns:

* Job Title
* Company
* Location
* Skills
* Experience Required
* Salary
* Date Posted

Note: The dataset used is a synthetic/sample dataset created for educational and practice purposes.

---

## Analysis Performed

* Data cleaning and preprocessing
* Handling missing and inconsistent values
* Exploratory analysis of job roles and skills
* Location-wise, experience-wise, and salary trend analysis
* Visualization of insights using charts

---

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Codeabc27/linkedin-job-trends-eda.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Open the notebook:

```bash
jupyter notebook
```
